bolt(php)

Usage

<?php
$http = bolt\http::start([
        'root' => __DIR__,
        'http.plugins' => [
            ['router', 'bolt\http\router']
        ]
    ]);

$http['router']->get('/', function(){
    $this->response
        ->setContent('Hello World');
});

$http->run();

Get Started

composer create-project boltphp/bolt project-name
cd project-name
php server.php
http://localhost:8000
